DDL，数据定义语言
  - 定义数据库对象(数据库，表，字段)；
  - 数据库表名通常以 `tb_ 开头 + 小写 + 下划线 + 单数名词`；字段命名以 `小写字母 + 下划线`
  - 数值字段：状态使用 tinyint；
  - 金额字段：通常使用 decimal 或 bigint；涉及国际汇率兑换使用 varchar；
  - 时间字段：年月日：date；时分秒：datetime； 


DML，数据操作语言
  - 增，`insert into ...`
  - 删，`delete from ...`
  - 改，`update ... set ...`


DQL，数据查询语言
  ```sql
    select 字段列表 from 表名称        -- 基础查询
    where 条件列表                    -- 条件查询
    group by 分组字段列表              -- 分组查询
    having 分组后条件列表 
    order by 排序字段列表              -- 排序查询
    limit 开始索引，每页条数;
  ```
