# project01-maven-junit

## Maven - 管理和构建Java项目的工具
- 配置 IDEA 的 Maven环境(全局)，如下步骤所示：
    1. 进入 IDEA 首页
    2. 【自定义】Customize ->【所有设置】All settings
    3. 【构建、执行、部署】Build,Execution,Deployment ->【构建工具】Build Tools
- 在当前项目中导入 Maven 模块；
- 配置依赖范围，如：`<scope>test</scope>`；
- 外部配置项目属性;
- `File > project Structure`，配置项目的“骨架”信息，包括使用的 JDK、模块、依赖库、编译输出路径等。

## JUnit 单元测试框架
- 测试类
  命名规范：`XxxxxTest`；
  可使用 `@SpringBootTest`修饰，表示加载完整的Spring应用上下文。
- 测试方法
  必须使用 `@Test` 修饰；
  命名规范：`public void testXxx`；
