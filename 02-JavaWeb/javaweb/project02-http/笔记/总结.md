# project02-http

## Tomcat - web服务器
- Tomcat 也被称为 Web容器、Servlet容器，服务的默认端口是 8080
- Servlet 用于处理 web 请求；
- Spring MVC 是基于 Servlet 实现的；

## HTTP
- HTTP 请求常见状态码
    - 1xx，响应中-临时状态码；
    - 2xx，响应成功；
    - 3xx，重定向；
    - 4xx，客户端错误；
        - 400，客户端请求语法错误
        - 403，无权限访问
        - 404，请求资源不存在
        - 405，405请求方式错误
    - 5xx，服务端程序抛出了异常；

- 程序中修改响应状态码
    - 基于 HttpServletResponse 对象来设置。
    - 基于 Spring 提供的 ResponseEntity 对象来设置。
